*A tag < a > vem de ancora, funciona com link.
* href é o atributo, após o =, se adiciona o endereco desejado, sempre entre aspas "".
* target é um atributo que é utilizado para abrir o link em uma nova aba da página.
* title é o atributo utilzado na mesma linha od link, para demonstrar a proxima ação do link.

* A tag img indica ixibir imagem, ela abre e fecha na mesma tag.
* O atributo SRC é o diminuitivo de source, ou seja, a fonte da informaçaõ que vai se acessar. Após o =, se adiciona o endereco desejado, sempre entre aspas "".
* WIDTH é um atributo para definir largura, o mesmo também pode ser usado dentro de outras tags.
*HEIGHT é um atributo para definir altura, o mesmo também pode ser usado dentro de outras tags.
* ALT="" é um atributo de alternativa em texto, caso o navegador não consiga carregar a imagem, atribuindo o que se refere a imagem.
